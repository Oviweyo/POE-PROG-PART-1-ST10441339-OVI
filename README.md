# NetSecureBotChat

## Cybersecurity Chatbot (C# Console Application)

NetSecureBotChat is a console-based chatbot developed in C#.
It is designed to provide basic cybersecurity awareness through interactive conversation with the user.

---

## Features

* Interactive chatbot conversation system
* User name input and personalization
* Cybersecurity-focused responses
* Input validation to prevent empty entries
* Modular code structure for maintainability
* Console-based user interface

---

## Topics Covered

The chatbot provides information on the following cybersecurity topics:

* Password security and strong password creation
* Phishing awareness and prevention
* Safe internet browsing practices
* General cybersecurity guidance

---

## Project Structure

* Program.cs – Application entry point
* Chatbot.cs – Core chatbot logic
* ConsoleUI.cs – Handles console output formatting
* ResponseHandler.cs – Processes user input and generates responses
* InputValidator.cs – Validates user input
* AudioPlayer.cs – Optional audio greeting feature

---

## Audio Feature

The project includes an optional audio greeting that plays when the application starts.

The audio file must be placed in the output directory:
bin/Debug/net8.0/

---

## How to Run

1. Open the solution file:
   NetSecureBotChat.sln

2. Build the solution using Visual Studio

3. Run the application

---

## Requirements

* .NET 8.0 SDK
* Visual Studio 2022 or later

---

## Purpose

This project was created for educational purposes to demonstrate:

* C# programming fundamentals
* Object-oriented programming principles
* Console application design
* Basic input handling and validation
* Simple chatbot logic implementation
